# 🌌 WpDev: Behavioural Intelligence Engine

> **Precision Reimagined.** A high-performance, real-time analytics suite designed to distinguish organic human interaction from synthetic bot behaviour using multi-vector forensic indicators.

---

## 🚀 The Propulsion Sequence (Landing)
The experience begins with a high-fidelity, 120-frame scroll-linked animation. As the user descends into the engine, the system's core architecture is revealed through dynamic text beats and seamless transitions from form to function.

## 📊 The Analytics Suite
Once the engine is assembled through scroll, the live **Behavioural Analytics Dashboard** activates:
- **Global Authenticity Score:** Real-time percentage of organic traffic density.
- **Timing Regularity Analysis:** Detects sub-millisecond mathematical repetition.
- **Engagement Bursts:** Identifies physical action speeds impossible for human actors.
- **Linguistic Consistency:** Evaluates interaction entropy and browsing intent.
- **Local Forensics Audit:** Instant, browser-side AI analysis of specific user identities.

## 🧠 Deep Tech Stack
*   **Frontend:** Next.js 15, TypeScript, Tailwind CSS, Recharts.
*   **Animation:** HTML5 Canvas + High-Density WebP Frame Sequencing.
*   **Backend:** Multi-threaded Python (v3.13) with `ThreadingMixIn` for non-blocking ingestion.
*   **Intelligence:** Logistic Regression inference model σ(w·x + b) calibrated for synthetic datasets.
*   **Safety:** Write-Ahead Logging (WAL) for crash-safe state persistence.

## 🛠️ Getting Started

### 1. Requirements
- Node.js 18+
- Python 3.10+
- `requests` python library (`pip install requests`)

### 2. Launch the Engine (Backend)
```bash
cd analytics_engine
python backend_python.py
```
*The engine will start listening on port **8082**.*

### 3. Stream the Data (Simulator)
```bash
# In a new terminal
python analytics_engine/csv_streamer.py
```
*This blasts the 446k-line `synthetic_bot_dataset.csv` into the engine at 5000x speed.*

### 4. Deploy the UI (Frontend)
```bash
cd wpdev
npm install
npm run dev
```
*Open [http://localhost:3000](http://localhost:3000) to begin the experience.*
